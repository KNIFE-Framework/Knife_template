# 🧰 3. Inicializuj Docusaurus

```bash
npx create-docusaurus@latest KNIFE_Template classic --typescript
cd KNIFE_Template
npm install
```

🧩 Vytvor potom tieto priečinky:

- `/docs/`
- `/docs/sk/`, `/docs/en/` – pre jazykové verzie
- `static/` – pre obrázky, HTML5, videá
